# BRAINOVA-GAMIFIED-WEBSITE-STUDY-PLANNER-AND-HABIT-TRACKER-
BRAINOVA is an intelligent study habit tracking web application designed to help students build and maintain effective learning routines. Built with Streamlit for an intuitive user interface, it combines data analytics, gamification elements, and machine learning to provide personalized insights and motivation.
Summary of the Project
BRAINOVA is a web application that makes tracking study 
habits fun. It helps students boost their productivity, stay 
consistent, and improve their overall learning experience. 
This is achieved by combining traditional study planning 
tools with game-like features. Many students today find it 
hard to stay disciplined and motivated while studying. 
This struggle often stems from a lack of a good
productivity tracking system with engaging interface. 
“BRAINOVA” solves this issue by offering a platform 
where users can set study goals, track their daily habits, 
monitor their progress, and stay motivated with features 
such as streaks, rewards, and visual progress tracking. The 
application uses a modular design, featuring a user-
friendly frontend and a well-organized backend that 
allows for efficient data management and growth. A 
standout aspect of the project is its use of Machine 
Learning components. These act as the system's "brain," 
studying user behaviour, including study patterns and 
performance trends, to offer valuable insights and 
personalized advice. These smart features let users spot 
areas for improvement, manage their time better, and 
develop good study habits over time. The project focuses 
on clean code structure and separates different functions. 
This design choice allows for real-time interaction, making 
the system both efficient and easy to expand. In 
summary, BRAINOVA seeks to change how students’
study by creating a more interactive, data-driven, and 
motivating experience. Its goal is to help students reach 
their academic targets in a more organized and engaging
way.
